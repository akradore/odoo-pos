Pos paid order invoice

Now in odoo there is no solution for make invoice of all paid order in a particular session. So we provide a module for convert all paid orders of a session in to invoice when session close and validate. In this module change all paid orders that have customer into invoice.

Screenshots

We can create pos order with customer.

Then payment can done with any payment method.

Now we can see all the paid orders in pos orders.

When session close and validate all the paid orders that have customer will convert into invoice .

we can see now all the paid order that have customer changed into invoice and remaining will be posted.

Select any invoice there we can see invoice in paid state.
